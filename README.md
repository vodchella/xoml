# XOml

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT) ![OCaml](https://img.shields.io/badge/OCaml-%23EC6813.svg?logo=ocaml&logoColor=white)

**XOml** is a very simple (easy-to-win) **Tic Tac Toe** game _(5 in a row, 10x10 board)_ implemented in [OCaml](https://ocaml.org/)

## Usage

With `dune`:
- `dune build` to build
- `dune exec xoml` to build and run
- `dune clean` to clean up build artifacts

With `make`:
- `make` to build
- `make run` to build and run
- `make clean` to clean up build artifacts
